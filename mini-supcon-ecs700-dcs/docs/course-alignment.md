# Course Alignment — SUPCON ECS-700 DCS

## Nine-School Curriculum Mapping

### MIT
- **6.302 Feedback Systems**: PID implementation, relay auto-tuning
- **2.171 Digital Control**: Discretization, scan-synchronized execution
- **2.14/2.140 Mechatronics**: I/O subsystem, signal processing

### Stanford
- **ENGR205 Process Control**: Cascade control, feedforward, distillation column example
- **AA272 GNSS**: IEEE 1588 PTP time synchronization
- **EE392 Industrial AI**: Health scoring, predictive maintenance

### Berkeley
- **ME233 Advanced Control**: Anti-windup, bumpless transfer
- **EE C128 Mechatronics**: I/O channel configuration, ADC/DAC scaling

### CMU
- **24-677 Advanced Control Systems**: Redundancy architecture, failover state machine
- **18-771 Linear Systems**: Signal scaling linear interpolation

### Georgia Tech
- **ECE 6550 Nonlinear Control**: Interlock logic, SFC execution
- **AE 6530 Optimal Estimation**: Signal filtering (EMA), CRC error detection

### Purdue
- **ECE 602 Lumped Systems**: Process point model, real-time database
- **ME 575 Industrial Control**: Boiler three-element control, BMS

### RWTH Aachen
- **Industrial Control Systems**: Full DCS architecture (German approach)
- **PLC/SCADA Engineering**: IEC 61131-3 SFC, MODBUS integration

### 清华 (Tsinghua)
- **过程控制工程 (Process Control Engineering)**: Reactor cascade, column control
- **运动控制 (Motion Control)**: N/A (DCS focus)
- **工业互联网 (Industrial Internet)**: OPC UA, SCnet architecture

### ISA/IEC Standards
- **ISA-88 Batch Control**: SFC batch sequencing in reactor example
- **ISA-101 HMI**: Process point display parameters (decimal places, EU labels)
- **IEC 61131-3**: SFC language implementation
- **IEC 61508**: PFD calculation, SIL verification
- **IEC 61511**: BMS interlock logic
- **IEC 62439-3**: HSR redundancy, failover time analysis
- **IEC 62443**: Domain security zones
- **IEC 60751**: RTD Pt100 linearization
- **IEC 62541**: OPC UA node model
- **IEEE 1588**: PTP time synchronization
- **NAMUR NE43**: 4-20 mA fault detection
