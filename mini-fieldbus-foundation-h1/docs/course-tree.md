## Learning Path

1. Physical Layer (IEC 61158-2) → Manchester, CRC-16, cable types
2. Data Link Layer (IEC 61158-4) → LAS, scheduling, Live List
3. Application Layer (IEC 61158-5/6) → FBAP, FMS, OD
4. System Management (FF-880) → SM agent, time sync
5. Device Types (FF-831, FF-103) → DD, CFF, ITK
6. Segment Engineering → power budget, health, commissioning
