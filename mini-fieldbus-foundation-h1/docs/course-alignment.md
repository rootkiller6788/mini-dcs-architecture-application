## Key Standards Covered

- IEC 61158-2 (Physical Layer) — Full cable specs, Manchester encoding, frame structure
- IEC 61158-4 (Data Link Layer) — LAS, token passing, VCR types, address assignment
- IEC 61158-5/6 (Application Layer) — FMS, Object Dictionary, function blocks
- FF-890 (Function Block Specification) — AI/AO/PID/RA algorithms, MODE_BLK
- FF-880 (System Management) — SM state machine, Find Tag, Set Address, Time Sync
- FF-831 (Device Description) — DD variables, methods, validation
- FF-103 (Common File Format) — CFF capabilities, ITK compliance
